 Formulário Inteligente com Validação em JavaScript 

Projeto se consiste na construção de um formulário com interatividade e validação robusta, utilizando JavaScript
1. Interatividade em tempo real:
    * Contador de caracteres dinâmico com limite (máx. 50).
    * Seleção dependente (em Cascata) para Estado e Cidade.
    * Efeitos de "focus" para feedback imediato no campo E-mail.
2. Validação avançada
    * Validação de email no blur e validação completa no submit.
    * Indicador de Força de senha (Fraca, Média, Forte).
    * Regras como mínimo de caracteres (Nome), campos obrigatórios (Select) e confirmação de Checkbox.
3. Check-up de processos 
    * Botão de enviar desabilitado até todos os campos estarem válidos.
    * Fornecimento de feedback visual, textual, incluíndo um "sumário de erros" unificado.
    * Lógica (JS) separada da interface (HTML/CSS).

Recursos utilizados: HTML5, JavaScript e CSS
